# Countries_Dashboard
Country dashboard to display information about countries and manage a wishlist 

This is a project from my object oriented programming module designed using Java and open source REST api (https://restcountries.com/) that displays information about different countries. In this application you can manage a simple wishlist, play a quick quiz game based on the informationi displayed and view different countries around the world. The main purpose of this project was to apply object oriented programming concepts, integrate a GUI framework with Java and interact with a REST api to retrieve meaningful information.
